----Для того чтобы работало корректно приложение нужно----

В строке Location = new Point(0,0) необходимо изменить значения, чтобы они совпадали с панелью управления(клавишами закрыть, развернуть, свернуть) исходной программы ARDOR.
ЕСЛИ же окна совпадают и не съезжают, то изменения здесь вносить не нужно.

!!!! ВНИМАНИЕ !!!!
В строке _ardorGamingProcess = System.Diagnostics.Process.Start("C:\Path\to\ArdorGamingAgile.exe") указать путь к ".exe" файлу вашей программы ARDOR.

После проведения всех этих манипуляций, можете спокойно пользоваться программой. Исходный код находится в файлах, но для этого вам нужна программа с компилятором для
редактирования и отладки исходного кода.

По сути, вы можете использовать данную наработку для сворачивания всех программ в трей, где такая возможность не предусмотрена производителем. 
Для этого вам придется в меню конструктора изменить иконку в строке notifyicon на подходящую, в остальном же, принцип прежний и описан выше.

Также на всякий случай я прикрепил .zip архив с изначальным кодом и содержимым программы, если вдруг, по какой-то причине программа не работает :)
